COGI — AI 코드리뷰 · 성장분석 플랫폼

팀 프로젝트 · Spring Boot + React · 2026 · 개발 진행 중

JavaSpring BootSpring SecuritySpring Data JPAJWTOAuth2TOTPRedisMariaDBReactTypeScript

개요

GitHub 웹훅으로 Pull Request를 받아 AI가 코드를 자동 리뷰하고, 반복적으로 발견되는 약점을 통계로 누적해 개인·팀의 성장 추이까지 보여주는 개발자 코드리뷰 SaaS입니다.

Spring Boot 백엔드와 React(Vite·TypeScript) 프론트엔드로 구성되며, 도메인별로 패키지를 분리하고 서비스는 인터페이스와 구현체를 나눠 결합도를 낮춘 계층형 아키텍처로 설계했습니다. 저는 인증/보안, 성장분석, 프론트엔드 연동을 중심으로 담당했습니다.

담당 파트 (Leeuswa)

01인증 방식 설계 (JWT + HttpOnly 쿠키)

  • 기존 토큰 방식을 HttpOnly 쿠키 기반 JWT 인증으로 전환해 XSS로부터 토큰 노출을 줄임
  • 로그아웃 처리와 인증 실패 시 401 응답 흐름을 일관되게 설계
  • 이메일을 유니크 키로 두는 계정 모델을 설계하고, 일반/소셜 회원 간 이메일 중복 검증 로직 정리

02소셜 로그인 · 2단계 인증(2FA)

  • GitHub·Kakao OAuth2 소셜 로그인 구현
  • TOTP 기반 2단계 인증(2FA)을 추가해 로그인 보안 강화
  • OAuth state 파라미터로 CSRF를 방어하고, 이미 연결된 계정과의 충돌 시 연동을 거부하도록 처리
  • GitHub 계정 연동 기능으로 사용자의 레포지토리와 서비스를 연결

03온보딩 · 약관

  • 소셜 로그인 최초 진입 시 온보딩 단계에서 약관 동의를 받을 수 있도록 흐름 구성
  • 약관 조회 API와 마이페이지 프로필·닉네임 설정 기능 구현
  • 인증 UX 개선 — 인증 코드 타이머·정지 문구, 에러 메시지 전달 정비

04성장 추이 대시보드

  • AI 리뷰에서 누적된 약점 통계를 바탕으로 팀/개인 단위 이슈 발생·해결 추이를 보여주는 대시보드 구현
  • 기간별 비교로 코드 품질이 개선되는 흐름을 시각적으로 확인하도록 설계

05주간 리포트 (스케줄러 + 메일)

  • 매주 월요일 오전 9시에 주간 리포트를 자동 생성하는 스케줄러 구현
  • 생성된 리포트를 HTML 메일로 발송하는 흐름까지 연동

06프론트엔드 연동

  • React(Vite·TypeScript) 화면과 회원가입·로그인·온보딩 API를 연동
  • 쿠키 인증 방식에 맞춰 프론트 인증 흐름을 연결하고, 빌드·타입 에러 및 누락 훅 정리

주요 기능 (서비스 전체)

  • 인증 / 보안 — 이메일 인증 회원가입, JWT 쿠키 로그인, GitHub·Kakao 소셜 로그인, TOTP 2FA
  • 코드리뷰 — GitHub 웹훅으로 PR 수신, 코드 컨텍스트 추출(JavaParser), AI 자동 리뷰
  • 성장 분석 — 약점 통계 누적, 성장 추이 대시보드, 주간 리포트 자동 발송
  • 결제 — Toss Payments 구독·크레딧 사용량·빌링 스케줄러
  • 관리자 — 회원·공지·약관·가이드라인 관리, AI 사용량 관리
