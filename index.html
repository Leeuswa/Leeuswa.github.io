소개

안녕하세요, Java와 Spring Boot 기반 백엔드 개발을 중심으로 REST API 설계, JPA, MariaDB를 활용한 웹 서비스 개발 경험을 쌓아온 이수환입니다. Controller–Service–Repository 계층 구조로 관심사를 분리하고, 인증/인가 처리와 데이터 흐름을 설계하는 데 관심이 많습니다.

단순히 기능을 구현하는 것을 넘어 동시성·인증·데이터 정합성 같은 '보이지 않는 안정성'을 고민합니다. 팀 프로젝트에서 예약 시스템의 동시 요청 문제를 비관적 락으로 해결하고, JWT·OAuth2 인증과 SSE 실시간 알림을 직접 구현했습니다. 최근에는 Spring AI, OpenAI API, RAG 구조를 학습하며 AI 서비스 개발로 역량을 넓히고 있습니다.
